# Multiple regional subdomains appear to have separate career pages but sitemap job counts are identical

## Problem
Multiple regional subdomains appear to have separate career pages but sitemap job counts are identical

## Solution
Compare sitemap job counts across regional subdomains before adding separate boards. If counts match exactly, the subdomains likely mirror the same global job pool. Use the primary domain (www) as the single board instead of adding redundant regional boards.
