# RSS monitor with generic preset returns 0 jobs on Teamtailor board because feed_url is not auto-detected

## Problem
RSS monitor with generic preset returns 0 jobs on Teamtailor board because feed_url is not auto-detected

## Solution
Use teamtailor preset with explicit feed_url: https://<company>.teamtailor.com/jobs.rss
