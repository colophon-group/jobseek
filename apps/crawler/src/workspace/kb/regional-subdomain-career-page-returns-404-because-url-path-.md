# Regional subdomain career page returns 404 because URL path uses localized language (e.g. /karriere/ in German instead of /career/ in English)

## Problem
Regional subdomain career page returns 404 because URL path uses localized language (e.g. /karriere/ in German instead of /career/ in English)

## Solution
Search for the company's career page in the local language (e.g. 'site:che.sika.com karriere'). Regional subdomains often use localized paths like /karriere/, /carriere/, /carreras/ instead of /career/. Check hreflang tags or search engine results to find the correct localized URL.
